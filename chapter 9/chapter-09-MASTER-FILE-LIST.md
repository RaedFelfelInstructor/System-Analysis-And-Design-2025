## Key Learning Outcomes

Students completing this chapter will be able to:

1. ☐ Explain UI vs UX and user-centered design
2. ☐ Create information architecture diagrams
3. ☐ Design wireframes at multiple fidelity levels
4. ☐ Apply Gestalt and visual design principles
5. ☐ Implement accessibility best practices (WCAG)
6. ☐ Create responsive, mobile-first designs
7. ☐ Build interactive prototypes
8. ☐ Conduct usability testing and heuristic evaluations
9. ☐ Document findings and prioritize improvements
